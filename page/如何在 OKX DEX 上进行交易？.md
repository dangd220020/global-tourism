   - 选择“创建钱包”，并点击“使用助记词创建钱包”来生成一个新钱包。
2. **启用生物识别认证**  
   - 您可以选择立即启用 Face ID 或稍后设置，以便通过生物识别功能访问钱包。

4. **备份钱包**  
   - 您可以选择“云端备份”或“手动备份”，务必妥善保管好您的助记词或私钥。请注意，一旦助记词丢失，将无法恢复您的钱包，任何获取该信息的人均可完全控制您的数字资产。
