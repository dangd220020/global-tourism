# DMIT 香港Lite VPS线路调整：电信联通优化取消，仅保留移动CMI直连

近期，DMIT对其香港Lite VPS产品线进行了重要调整。由于持续遭受DDoS攻击，DMIT宣布取消HKG.Lite系列对中国电信和联通的线路优化，仅保留移动用户的CMI直连线路。这一变动在用户社区引发了广泛讨论。

## 线路变更详情

根据官方公告和用户实测数据：

- **电信/联通用户**：去程路由改为经DMIT圣何塞节点中转至香港，回程走联通AS4837线路
- **移动用户**：继续保持双向CMI直连优化
- **服务承诺**：DMIT明确表示Lite系列不提供线路质量保证，这与Pro版本的服务条款有本质区别

值得注意的是，HKG.Lite的三网回程CMI优化仅维持了约两个月时间。DMIT解释称此次调整受"多重因素"影响，其中持续的网络攻击是主要原因之一。

## 其他政策调整

除线路变更外，DMIT还做出了以下调整：

1. 取消新用户的Guarantee+选项（原可确保获取未被墙的IP）
2. 推出临时补偿政策：对11月7日至12月7日期间在二手交易中受骗的HKG.Lite.TINY用户，提供等值SJC系列产品作为补偿

## 用户反馈与品牌影响

此次调整在HostLoc论坛和Telegram社群引发热议，主要争议点包括：

- 黑五促销后突然变更服务的时机问题

DMIT承认事件对其声誉造成影响，但强调Lite系列从未承诺过线路优化保障。为缓解用户不满，公司推出了针对特定情况的补偿措施，同时明确表示这些举措"仅出于礼貌"而非义务。
