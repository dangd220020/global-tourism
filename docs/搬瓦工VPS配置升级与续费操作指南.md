# 搬瓦工VPS配置升级与续费操作指南

## 一、KVM架构VPS升级流程

对于使用搬瓦工新版KVM架构VPS的用户，可通过以下步骤完成配置升级：

1. 登录后台管理面板
2. 进入Support → Tickets → Open new ticket
3. 使用翻译工具将升级需求转换为英文提交
4. 客服会提供专属支付链接
5. 完成付款后系统自动升级配置

## 二、自助升级功能详解

搬瓦工VPS提供便捷的自助升级服务：

1. **进入管理界面**：Services → My Services → Manage
2. **选择升级选项**：点击Management Actions → Upgrade/Downgrade
3. **选择目标套餐**：系统会自动计算差价
4. **完成支付**：无需手动迁移数据，配置自动更新

> 重要提示：目前仅支持套餐升级服务，暂不支持配置降级

## 三、VPS续费操作说明

- 系统会在服务到期前7天发送续费提醒邮件
