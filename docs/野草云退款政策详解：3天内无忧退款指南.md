2. **操作步骤**：
   - 登录野草云客户中心
   - 提交服务单至售后或财务部门

## 退款服务条款细则
1. **适用产品**：仅限自营虚拟主机/VPS/云服务器（购买后3天内）
2. **不适用产品**：
   - 独立服务器
   - 域名注册
   - AWS Lightsail
   - 其他特殊服务项目
3. **批量购买规则**：一个月内多次购买同类型产品申请退款，将扣除10-30%费用
4. **特殊情况**：
   - 超过退款期限不予退款
   - 违反服务条款不予退款
   - 特价促销产品（页面明确标注）不予退款
5. **账户规则**：
   - 账户余额不支持退款
6. **行为规范**：
   - 辱骂客服人员将丧失退款资格
   - 未经沟通公开诋毁野草云声誉将立即封号且不予退款
