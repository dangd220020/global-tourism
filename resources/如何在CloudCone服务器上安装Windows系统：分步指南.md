## 第一步：登录Linux VPS
1. **使用SSH客户端连接**
   - 推荐使用Bitvise等专业SSH工具
   - 输入主机地址、用户名和密码进行登录

2. **获取终端权限**
   bash
   sudo -i
   
   - 此命令可将普通用户切换为root权限
   - 成功登录后保持终端窗口开启

## 第二步：获取并执行安装命令
1. **获取安装密钥**
   - 访问TinyInstaller官网获取有效许可证
   - 登录后复制对应的安装命令

2. **执行安装流程**
   - 将复制的命令粘贴到终端窗口
   - 按Enter键开始执行

3. **选择系统镜像**
   
   输入对应数字选择Windows版本：
   1. Windows Server 2012
   2. Windows Server 2019
   3. Windows Server 2016（推荐）
   
   - 输入数字后确认选择

4. **跟踪安装进度**
   - 复制生成的跟踪链接
   - 输入"y"确认开始安装
   - 通过跟踪链接实时查看安装进度

## 第三步：连接Windows系统
1. **使用远程桌面连接**
   - 打开Windows自带的远程桌面工具
   - 输入从网站获取的访问信息

2. **首次登录配置**
   - 建议立即修改默认密码
   - 安装必要的驱动和更新

## 常见问题解答
- **安装时间**：通常需要15-30分钟
- **网络要求**：确保VPS有稳定的网络连接
- **镜像选择**：根据实际需求选择合适版本
